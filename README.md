# [C# 코딩] 나의 키오스크 만들기

## 개요
- C# 프로그래밍 학습
- 1줄 소개 : 
- 사용한 플랫폼 : C#, .NET Windows Forms, Visual Studio, Github.
- 사용한 컨트롤: RadioButton, CheckBox, ListBox, Label, Button, GroupBox.
- 사용한 기술과 구현한 기능 : 메뉴 선택, 주문 내역 표시, 총 금액 계산, 초기화 기능.
